裸眼3D营销视频 — AI 生成 + 后期合成方案调研

结论前置

用 AI 生成裸眼3D营销视频的核心逻辑：AI 负责生成动态素材，后期负责制造"冲出屏幕"的视觉欺骗。整个流程分 5 步，不需要 3D 建模能力，1-2 小时可出一条。

关键发现： - 裸眼3D 效果的本质是画框 + 出框 + 阴影三要素的组合 - 2025 年已有一键工具（Somake AI）可直接从图片生成裸眼3D 动画，零门槛 - 专业级效果需要 AI 视频生成 + CapCut/AE 后期遮罩，5 步完成 - 社交媒体短视频（竖屏）比大屏广告牌更容易实现——因为观看视角是固定的

一、核心原理（30 秒理解）

裸眼3D 不是真的 3D，是利用画框边界制造的深度错觉：

┌──────────────────────────────┐
│                              │
│      画框内 = 屏幕内          │  ← 观众大脑认为这是"屏幕平面"
│                              │
│         ┃  产品/角色          │
│         ┃  冲出画框           │  ← 超过画框边界 = "破屏而出"
│─────────╋────────────────────│
│         ┃                    │
└─────────╋────────────────────┘
          ┃  产品的一部分
          ┃  在画框外面          ← 大脑判定"物体在屏幕前方"
          ┃
          阴影落在画框边缘上      ← 加强空间暗示

三要素缺一不可： 1. 画框（黑色/白色边框）— 告诉大脑"屏幕在哪里" 2. 出框（主体突破边框边界）— 制造"物体飞出"的感知 3. 阴影/遮挡（物体在画框上投射阴影）— 加强深度暗示

二、AI 生成 + 后期合成：完整 5 步流程

Step 1：准备产品/主体素材

目标：获取产品或主体的动态视频素材。

素材类型 推荐工具 操作
产品动态展示 Kling 3.0 / Seedance 2.0 输入产品图 + prompt "产品旋转/飞出/展开"
角色/人物动作 Runway Gen-4.5 / Kling 3.0 图生视频，生成人物向镜头冲来的动作
纯文字生成 即梦 AI / Kling 3.0 文生视频，描述"产品从黑暗中飞向镜头"

Prompt 写法关键：

✅ 好的 prompt：
"[产品名] flying towards camera, dramatic lighting, dark background, 
shallow depth of field, cinematic slow motion, front-facing angle"

"产品从画面深处飞向镜头，暗色背景，浅景深，电影级灯光，慢动作"

❌ 避免：
- 不要生成带固定背景的素材（后期难抠）
- 不要用复杂场景（干扰出框效果）
- 背景尽量纯色/渐变/暗色

输出要求： - 分辨率：至少 1080p（推荐 2K） - 时长：3-8 秒（一条裸眼3D视频最佳长度 5-15 秒） - 背景：纯色或简单渐变（方便抠图）

Step 2：AI 抠图/分离主体

目标：将产品/主体从背景中分离出来，得到带透明通道的素材。

工具 方式 适用场景
CapCut 智能抠像 画中画 → 智能抠像 → 自动去背景 手机端快速操作
Runway "Remove Background" 上传视频 → 自动去背景 视频级逐帧抠图
After Effects RotoBrush 3.0 AI 自动追踪边缘 专业级精细抠图
剪映 "智能抠像" 导入素材 → 智能抠像 国内用户最快方案

如果 Step 1 用了暗色/纯色背景，这步会非常简单。

Step 3：构建画框（核心步骤）

目标：创建一个"屏幕边框"作为深度参照。

方案 A：CapCut / 剪映操作（最简单）

  1. 新建项目，画布比例选 9:16（竖屏）或 16:9（横屏）
  2. 添加纯色背景层（黑色或深灰色）
  3. 添加白色/浅灰色矩形作为"画框"：
  4. 尺寸：比画布小一圈（上下左右各留 40-60px 黑边）
  5. 圆角：8-12px（模拟真实屏幕感）
  6. 将画框层放在最下方，产品素材放在画框层之上

方案 B：After Effects 操作（更精细）

  1. 新建合成（1080×1920 竖屏 或 1920×1080 横屏）
  2. 创建 Shape Layer → 圆角矩形，填充白色，大小比画布小 80px
  3. 产品素材放在画框之上
  4. 使用 Track Matte（遮罩）控制哪些部分在"框内"、哪些在"框外"

画框设计原则： - 画框必须有可见的边界（边框线或颜色对比） - 画框占画布面积 75-85%（留出足够的"出框空间"） - 画框颜色与背景有明显对比（白框+黑底 或 金属质感框+深色底）

Step 4：制作出框遮罩（灵魂步骤）

目标：让产品的一部分"突破"画框边界，创造 3D 出框幻觉。

CapCut / 剪映具体操作：

  1. 将产品视频放在"画框层"上方
  2. 选中产品层 → 蒙版 → 镜面蒙版 / 矩形蒙版
  3. 关键帧动画：
  4. 第 0 帧：蒙版完全在画框内（产品被画框裁切）
  5. 第 X 帧：将蒙版向外扩展，超过画框边界

  6. 效果：产品在动画中逐渐"冲出"画框

  7. 在画框边缘添加遮挡层：

  8. 画框的顶部/底部边线需要遮住产品的一部分
  9. 例如：产品从画框顶部飞出，但画框底部仍然遮住产品底部
  10. 这个"部分遮挡"是出框效果的关键！

After Effects 操作：

  1. 将产品层放在画框层上方
  2. 创建一个 Mask（形状为画框上半部分），设为"None"
  3. 产品层添加 Track Matte → 使用自定义遮罩
  4. 产品的上半部分（出框区域）不受遮罩限制
  5. 产品的下半部分被画框遮挡 → 产生"一半在框外、一半在框内"的深度感

出框方向选择： | 出框方向 | 适用场景 | 效果强度 | |----------|----------|----------| | 顶部出框 | 产品向上飞起 | ★★★★ | | 底部出框 | 人物从画框下方走出 | ★★★★★（最强） | | 左右出框 | 产品横向伸出 | ★★★ | | 多方向出框 | 爆炸/碎片飞出 | ★★★★★ |

Step 5：添加阴影与光效（增强真实感）

目标：用阴影和光效强化"物体真的在屏幕前方"的感知。

  1. 投射阴影：
  2. 在产品"出框"的部分下方，添加一个模糊的黑色阴影
  3. 阴影要落在画框的边缘线上（不是背景上）
  4. CapCut：添加贴纸 → 黑色椭圆 → 高斯模糊 → 降低不透明度到 30-50%

  5. AE：Drop Shadow 效果 → 调整距离和模糊度

  6. 边缘光/环境光：

  7. 在产品出框的边缘添加微弱的光晕

  8. 模拟产品被"真实环境光"照亮

  9. 景深效果：

  10. 画框内的背景添加高斯模糊（2-4px）
  11. 出框的产品保持锐利

  12. 对比产生前后层次感

  13. 运动模糊（可选）：

  14. 如果产品有快速运动，添加适度的运动模糊
  15. 增强动感和冲击力

四、进阶技巧

1. 深度图强化法

用 AI 深度图工具（Depth Map Generator）为产品图生成深度信息，然后在 AE 中用深度图驱动 2.5D 视差动画：

  • 工具：Depth Anything V2 / Marigold / Boris FX Continuum
  • 流程：产品图 → AI 生成深度图 → AE 中用深度图做视差位移 → 前景突出、背景压缩
  • 效果：比简单的出框遮罩更自然，有真实的空间透视感

2. 多层出框法

将画面分为 3-5 层，每层设置不同的出框程度： - 第1层（最远）：完全在画框内 - 第2层（中间）：边缘轻微出框 - 第3层（最近）：大幅出框 + 强阴影

3. 镜头运动强化

在最终合成时，给整体画面添加轻微的晃动或推拉： - 画框固定不动 - 产品随镜头运动有微小的视差偏移 - 增强3D空间感

4. 音效配合

出框瞬间配合"嗖"的音效或重低音，多感官叠加强化 3D 冲击感。

六、避坑指南

常见错误 为什么错 正确做法
没有画框直接做 没有参照物，大脑无法感知深度 必须有可见边框
整个产品都在框外 失去"突破"的对比感 保留 40-60% 在框内
阴影方向错误 破坏空间逻辑 阴影落在画框边缘线上
背景太复杂 干扰出框主体的视觉焦点 用纯色/渐变暗色背景
视频太长 效果感知疲劳 5-15 秒最佳
出框部分模糊 近处物体应该更锐利 出框部分必须清晰锐利
