我们会介绍，我们是北大唯一的学生人工智能社团，社团做过的活动（大模型闭门分享、动手做agent、训小模型的共学课、论文共读、研报共写、开发者workshop…），以及我们社区的开发者和探索者，还有这次来WAIC的OPC大赛获银奖的项目。屏幕上轮番播放两年来活动的各种照片，但更多维度的信息很难一下子传达。何况观众都逛的口干舌燥。

说实，展会熙熙攘攘，更多是来链接资源的人，希望认识人才、看机会&信息差、渴望接入网络的人。大家都时间紧凑、期待高。而开发者、研究者的数量相比观众少很多。

有一个在大模型做算法的朋友逛完展馆说：我这个科研人来这里觉得确实热闹，但还不如在实验室多看点论文、写点代码。

我不太同意他的看法，其实埋头做科研和了解真实市场热点是不冲突的，并非非此即彼的关系。确实，这里不会马上找到论文搭子、硬核实验室内部分享的一些观点。因为大家还是需要一起做事才能建立真信任，才会有更深入的磨合与合作，才敢说一点非共识的信息。

而更深的信任，要看利益冲突后、意见不一致时，对方言语之外的行为——是重视组织利益边界，还是更有私心。到底有没有担当。

时间，才是最好的答案。

而我们社团这两年多所沉淀的，就是时间积累，所带来的信任。因为研报得一篇篇写，共学营得一期期办，活动得一场场做。

没有什么捷径可言。

这样踏实的气质，吸引的更多是一帮同样有点理想主义、也愿意实操的人。也正是这样的调性，让我们结识了同气相求的合作企业和嘉宾。一些真诚的闭门分享和线下workshop，来的都是互相筛选过的人。

有些追求学生官僚主义，或者希望快速流量变现的人，在我们这里待着会很别扭。当然世间法嘛，各种路子都有人走，也可以阶段性拿一些结果。但我还是觉得保持纯粹、坚持组织的价值观和边界，做更长线的事情，才是最优解。

随着未来社团进入新的阶段，随着规模扩大，这样纯粹的气质和价值观，能否继续传承，是个很大的挑战。我也没有想好～

AI 让“干中学”成为了最聪明的学习方式。

以前想学点东西，总想着——

“先看几门课”“先弄懂原理再说”。

结果一拖，就是几个月。

但现在有了 AI，学习的顺序彻底反了：

不是「学会再干」，而是「干着干着就会」。

我自己的经验是👇

用几个 AI 工具辅助学习与实践：

1️⃣ 用 ChatGPT 做“速查＋信息搜集 + 答疑解惑”

  • 卡住一个概念、一个函数，就直接问 ChatGPT。

  • 想做产品原型和项目规划，先通过对确定demo。

2️⃣ 用 NotebookLM 做“资料整理＋深度理解”

  • 我把报告、PDF、设计文档往里一丢，它帮我提炼重点、生成思维导图。

  • 从“看懂”→“抓住重点”→“输出”路径缩短很多。

3️⃣ 用 Claude Code 做“自动化任务＋代码生产”

  • 如果项目涉及代码，用 Claude Code 这个强大的 AI 编程助手开发，能很大程度加快开发速度。

  • 还可以用它处理一些自动化任务，帮助我做低质量重复性的工作。

4️⃣ 用 Figma 做“设计＋可视化”

  • 想把想法画出来、结构梳理，直接在 Figma 提出 prompt，生成草图、流程图。

5️⃣ 把它们串起来：做 → 查 → 反思

  • 做：先选一个小项目／写段代码／画个设计稿。

  • 查：遇到障碍，用 ChatGPT 查；整理资料用 NotebookLM；代码用 Claude Code；设计用 Figma。

  • 反思：结束后，用 AI 帮你回顾、总结、提炼下一步行动。

说实，AI 让“学习”变廉价了，很多偶尔用到一次的内容或许不必再系统学习。

但是如果你对某一方面有较高的要求 / 专业性，那么你还是要踏踏实实学基础、原理，AI 目前还只是提效的工具，或者说是第二大脑。

1⃣ 盘整必出方向：不急，就是优势

高位横着走，不追。

低位磨来磨去，不割。

市场没给方向，就是不给机会。

这时候最赚钱的动作——不动。

横盘期最容易让人手痒。

短线不是比操作频率，而是比谁更能忍。

跌深进，涨稳出。

听上去简单，做起来难。

大部分人在恐慌时砍在地板上，在亢奋时买在天花板。

你只要反着来，就已经超过很多人。

4⃣ 暴跌往往才是机会窗口

慢跌磨情绪，快跌砸结构。

但快跌后往往才有高质量反弹。

真正的机会不是长得慢、跌得慢，而是情绪断层时。

底部区域不 All in。

每跌 10% 加一点，就是在给自己拉低成本、增加利润空间。

这是小资金把风险压到最低的方式之一。

6⃣ 一旦变盘，要快、要狠、要干净

暴涨横盘 → 先走本金，只留利润试错。

暴跌横盘 → 别硬扛，第一时间切割亏损。

短线是快节奏游戏，动作慢了就是亏。

一句总结：

不猜、不追、不赌。

你只要守规则、控情绪、稳执行，就已经领先绝大多数人。
