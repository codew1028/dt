html-anything 安装方案

面向 AI Agent 的多模板 HTML 生成器 — OpenClaw workspace-eight 技能安装文档

项目概述

html-anything 是一套面向 AI Agent 的多模板 HTML 生成器，来源于 nexu-io/html-anything。安装为 OpenClaw workspace-eight 的 Skill，让小八具备从 75 个设计级模板中选择并生成精美单文件 HTML 的能力。

INFO
75 个模板涵盖 13 个大类，覆盖从文章排版到视频帧生成的完整场景。所有模板均为单文件 HTML，零外部依赖。

安装位置与结构

安装后的目录结构如下，所有文件集中在 workspace-eight/skills/html-anything/ 下：

text
workspace-eight/
├── skills/
│   └── html-anything/
│       ├── SKILL.md            ← 技能入口文档
│       ├── catalog.json        ← 75 条模板索引
│       └── templates/          ← 75 个子目录
│           ├── article-magazine/
│           │   ├── SKILL.md
│           │   └── example.html
│           ├── deck-pitch/
│           │   ├── SKILL.md
│           │   └── example.html
│           └── ...（共75 个）
├── logs/
│   └── html-anything/         ← 产出本地目录
└── TOOLS.md                   ← 已同步更新
WARNING
每个模板子目录必须同时包含 SKILL.md 和 example.html 两个文件，缺失任意一个将导致该模板不可用。

交互流程

Boss 说“做 HTML”时，小八会按以下流程执行：

Boss 说“做 HTML” → 三选一路由 → 大类选择 → 具体模板选择 → 读取模板 SKILL.md → DeepSeek-V4-Pro 生成 → 保存到 logs/ → 上传 Google Drive
DANGER
三选一路由指的是：md-to-html（aihot 风格）、note-slides（笔记幻灯片）、html-anything（75 模板）。Boss 未指定时必须先问用哪个 Skill。

执行配置

配置项值
执行模型DeepSeek-V4-Pro（sessions_spawn 指定）
产出目录logs/html-anything/
命名格式YYYYMMDD-{模板名}-{标题摘要}.html
Drive 样例GX/HTML模板样例
索引文件skills/html-anything/catalog.json
