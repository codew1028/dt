🔥《零基础让Claude学会抓取任何网站实时数据从此告别幻觉数据》

（5分钟设置，10倍提升你的AI能力）

如果你是AI完全新手，从来没写过程序，却想让AI帮你抓取真实网站数据（找客户、查竞品、看趋势），这篇教程就是为你量身打造的。

第一部分：准备工作（10分钟，缺一不可）

工具要求（超简单）：

  • 电脑（Windows/Mac都行）

  • Node.js 20.6 或更高版本（去 http://nodejs.org 下载安装，装完终端输入 node -v 检查）

  • 一个免费Apify账号

步骤1：注册Apify并拿密钥

  1. 打开浏览器，访问 https://apify.com

  2. 点击“Sign up”，用邮箱或Google注册（完全免费）

  3. 登录后，直接点这个链接：https://console.apify.com/account/integrations

  4. 点击“Create new token” → 随便起名 → 复制那串长长的密钥

步骤2：保存密钥（安全第一）

  1. 在电脑新建文件夹，叫“my-ai-project”

  2. 里面新建文本文件，改名为 .env（前面必须有个点）

  3. 打开文件，粘贴下面内容（把your_token_here换成你的密钥）：


APIFY_TOKEN=your_token_here
  1. 保存。以后所有项目自动读取这个密钥，千万别把密钥发给别人或写进代码。

步骤3：全局安装辅助工具

打开终端（Windows搜cmd，Mac搜Terminal），复制粘贴运行：


npx skills add apify/agent-skills

跟着提示安装mcpc CLI（点Yes即可）。这步只做一次。

第二部分：激活技能（Claude Code专用，2分钟）

  1. 打开Claude Code（Anthropic官方终端AI编程助手），进入你的项目聊天窗口

  2. 输入命令并回车：


/plugin marketplace add https://github.com/apify/agent-skills
  1. 再输入：

/plugin install apify-ultimate-scraper

安装成功后，Claude会回复“技能已激活”。
