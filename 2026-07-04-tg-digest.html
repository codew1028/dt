周末我请了一个做AI的朋友，来公司交流。

我们希望把AI真正融进方法论里，不是单独教工具。这个过程里，我会不停找身边懂的人来聊，让他们直接挑毛病，看看我们哪里没想清楚。

这一聊就是几个小时。

基本没停，中午饭直接点外卖，在会议室吃。信息量特别大，神经一直绷着，要听、要记、又要思考。结束的时候，人很累但脑子是兴奋的。

我突然意识到：

像我这种在互联网行业混了这么多年的人，一直有个很隐蔽的焦虑，就是怕被淘汰。

尤其是这几年，行业越来越年轻化，拼反应、拼节奏、拼新东西。

很多工作，二十六七岁的人就能干，而且干得不比你差，你心里是清楚的。

这种焦虑，35岁以后的人，基本都有。

但AI出来之后，我第一次有了不一样的感觉。

不是说我突然变年轻了，而是我发现：AI并不是在抹平所有人的差距，它反而在重新拉开差距。

比的是：谁真的有货。

什么叫有货？

对用户的理解、对内容的判断、对行业的认知、你见过多少项目、你踩过多少坑，以及你的社会阅历，都算。

AI可以把这些东西放大，这就是老炮儿的机会。

我想起很多年前的一件小事。

那会儿在美团，有朋友给我看一篇公众号文章，说这篇写得真好，真美。

我当时的第一反应是：哪儿美了？不就是一篇文章吗。

现在回头看，那时候我是真的不懂内容。不是不聪明，还是专业度不够，没货。

后来我在平台做运营，尤其是在短视频平台，每天都在判断内容：什么是好内容、什么是不好的、为什么这个能火...

看多了，拆多了，你慢慢就有感觉了。

现在我再看一条内容，不管是视频还是公众号，我基本都能拆开来看：

它的结构是什么、前几秒的信息量在哪里、节奏怎么走、哪些地方在拉你停留、哪些地方在浪费时间...

这些东西，在我这儿已经不是感觉，而是能说清楚的东西。

你真的用AI的时候，你会发现，这个能力变得异常重要。

人不写初稿，AI不写终稿。

你必须反复跟它对，但问在于，如果你自己不知道“好内容长什么样”，你根本不知道该怎么继续问。

你只会觉得不对，但说不出来哪里不对。你也不知道该怎么改，只能让它再来一版。

最后的结果就是，越改越像AI。

比如我们有一个关于“个人优势”的工具，我以前一直觉得已经想得挺清楚了。

但当我想把它变成AI智能体的时候，问一下全冒出来了。

专业到底从哪拆；

拆出来的几块怎么排序；

如果某一块特别弱，其他部分要不要补；

这些之间有没有逻辑关系...

我发现，有些东西我以前是“模糊正确”，但并没有想透。

AI会反过来逼你，你不说清楚，它就给不了你答案，这就是在训练你的逻辑能力，考验我们的是：

有没有能力把一个问拆清楚，能不能用准确的语言描述你的想法，能不能判断对方给你的结果是不是对的。

这对我触动很大。

作为一个文科生，我的机会终于来了，能干点大事了。

年轻的时候，我看过很多创业故事，硅谷的也好，国内的也好，你会发现那些人几乎都是程序员。

盖茨、小扎、马化腾、李彦宏、张一鸣...

他们能搞大事，和掌握了代码这个工具是有关系的，能把想法落地。

但我不行，我是运营呀，只能把已经创作出来的产品，想办法让它运转的更好，基本不参与创造的部分。

有了AI，我意识到，不会写代码的人，有了和算法对的机会，也能干大事了。

但这个对，对能力是有要求的：

考验的不是技术，是表达和逻辑思维能力。要把问讲清楚，要用精确的语言描述需求，要判断AI给你的东西是不是你想要的...

这些能力，恰恰是我在大厂这些年，被文档、被讨论、被反复挑战训练出来的。

讽刺的是，之前自己不具备这些能力，也觉得没啥用。

当年因为文档里的一句，在老板周会上被挑战了20分钟。当时觉得无聊至极，现在看，这个能力终于能用在正经地方了。

你之前不会的东西，AI帮不了你。

这也是我现在反复提醒客户和会员的。

不要被外面那些“一键生成脚本的智能体”所误导，你都不懂啥是爆款内容，有啥神仙的智能体也不好使。

别把结果和前提搞反了。

你得先知道什么是好选，什么是被验证过的爆款结构，为什么这个内容能火，AI才能帮你提效。

有人会觉得，我说的这些所有人都懂，不能称之为“机会”。

真的不是！

道理都懂，但真正去做的是少数；用正确的方式去做的，就更少了；能持续用正确的方式去做的，少之又少...

所以，现在就开干吧！

六月，Firsto 挺过了暑期淡季

六月对 Firsto 来说，不好打。

刚开始就连续 5 天零订单。

每天打开后台，访问量还在，但就是没人下单。

那几天我心里其实有点发毛。明知道平台型产品收入会波动，但真遇到连续鸭蛋，还是会忍不住想：是不是哪里出了问题。

后来想明白了：国外已经开始放暑假。

整个海外市场进了"夏日淡季"，这种时候能把基本盘守住就不算差。

好在月底回了一点血。

靠着一篇 Guest Post 和一个产品 Review 的进账，六月收入最终落在 860 刀。

比五月大概少了 5%。

我对这个结果能接受

整个六月，Firsto 几乎没做什么新功能。

不是没想法，是越来越觉得：对一个已经跑起来的产品发布平台来说，功能没那么缺。

更缺的是内容、流量，还有被看见的机会。

淡季里客户还愿意为 Guest Post 和评测买单，本质上买的是平台的长期权重。

所以七月我大概率不会继续猛加功能了。

我想把时间更多投到内容建设上。

买了个新域名，试水新生态

六月还是没忍住，又买了一个新域名。

不过这次不是再挖一个大坑，而是给自己搭了个练习站。

本质上是一个 Google TTS 的套壳产品。

我真正想练的是整套技术栈：

Cloudflare Workers

Cloudflare D1

Cloudflare R2

以前我一直用 Next.js 部署。

这次算是第一次把一个小应用完整跑在 Cloudflare 的生态里。

体验比我预想的顺手。

部署简单、成本低，对这种轻量 Web 应用来说确实舒服。

更意外的是，这个几乎没做推广的小站，已经来了 3 个注册用户。

当然，营收还是 0。

我现在的想法很直接：哪天真出现第一个付费用户，我再开始认真做 SEO，把关键词一点点铺开。

在没有收入之前，它就先当我的实验田。

但有一点我基本确定：这套技术栈以后会成为我做 Web 产品的默认选择。

也放弃了一些东西

去年六月，我一口气买了四个域名。

一年过去。

直到续费提醒弹出来，我才发现自己已经完全不想继续做它们了。

最后一个都没续费。

以前我总觉得，域名一年也没多少钱，先留着，万一哪天用得上。

我觉得对群响、生财和程前社群等商业模式的思考，恰恰解答了刘思毅为啥会被香港保险吸引，表面上看，这似乎只是一个千万级网红为了“身份续签”而做出的现实妥协，其实是他试图将社群的“虚胖流量”转化为“硬核商业资产”的一次努力。

当下商业社群的三大病灶：

1，交付沉重，深陷体力活：社群高度依赖创始人个人的内容输出（如常年直播、日更），交付物是信息、情绪和人脉。这种非标交付极度消耗创始人精力。

2，中心化依赖，缺乏组织杠杆：创始人是唯一的发动机，停更一天活跃度就下降。没有建立起去中心化的网络，收益天花板极低。

3，客群下探，深陷生存焦虑：服务“想月入2-5万”的群体，客单价被死死压在几百几千元。只能靠情绪消费和流量套利，无法提供高维价值。

从公众号到视频号红利，社群1.0时代已经见顶，“情绪和资源的时间贴现”终将到期。破局之路在哪？香港保险恰好交出了一份满分答卷。

一，完善的产品交付：从“情绪安慰”到“金融契约”

社群的痛点在于“交付无底洞”。刘思毅需要天天编新词、造金句，稍有不慎就会被贴上“割韭菜”的标签，因为交付质量全凭主观感受。

港险的解药在于“标准化履约”。保险是一套极度成熟的金融产品，背后有全球顶级金融机构的信用背书。用户买的不是一节课或一个社群门票，而是一份具有法律效力的契约，收益白纸黑字写在合同里。

这标志着从“卖信息/情绪”到“卖履约”的跨越。刘思毅无需再为后端交付质量焦虑，他只需要做好前端的信任构建，繁重的交付工作由保险公司接管。他终于可以从“高级个体户”的体力活中抽身。

二，去中心化组织：从“单人引擎”到“网络杠杆”

社群的痛点在于“中心化IP依赖症”。群响再大，也是刘思毅一个人的独角戏，无法实现真正的组织裂变。

港险的解药在于“代理人基本法”。保险行业的底层逻辑是一个天然去中心化、可裂变的组织架构。刘思毅完全可以利用自己的IP势能，招募合适人成为他的合伙人。

此时，他不再是单打独斗的内容生产者，而是变成了一个“资源路由器”和“团队长”。他的收入不再仅仅与个人的直播时长挂钩，而是与他搭建的代理人网络深度绑定。这实现了真正的组织杠杆，让别人的时间为他赚钱，彻底打破了社群的单人增长瓶颈。

三，高客单价客群：从“生存焦虑”到“资产配置”

社群的痛点在于“客单价极低”。为了做到几千万营收，需要疯狂拉新，流量成本极高，且服务的是处于生存焦虑中的低净值人群。

港险的解药在于“向上兼容”。香港储蓄险、重疾险的客单价极高（动辄几万、几十万美金），瞄准的是有财富传承、资产隔离、海外配置需求的高净值人群。

向10万个焦虑青年卖999元的课，不如向100个企业主卖10万美金的保险。

用保险的标准化产品交付解决体力透支，用保险的代理人网络架构解决中心化依赖，用保险的高客单价金融属性实现客群价值跃升。

这里也可以把「保险」换成其他有同样效果的产品。

这也给所有依赖个人IP的社群主提供了建议：单纯的流量生意终有尽头，只有将流量导入具备真实履约、网络杠杆和高客单价的业务中，社群IP才能完成从“网红”到“商人”的惊险跃升。

上游发行出品是一种能力；

中间做重建是一种能力；

下面做运营投放又是一种能力。

你要知道：他用了多少流量做到月入10万？

每一个加V用户产生多少利润？

后端转化率是多少？

复购有没有？

后端机构给他的结算方式是什么？

这个模式能不能复制到别的城市？

5 对业务的清晰很重要。很多时候业务不断堆积累加，变成了一个培训公司➕投流公司➕人力资源管理公司，这就不对了。链路太复杂了。

第一个，你得先判断这个赛道大还是不大。

我们能不能干。

他打第一家店的时候，那个店就不是一个普通小店。那家店从一开始，就是为未来1万家店做准备的。

标准化程度有多少？

供应链储存难度有多少？

上游集采怎么做？

YouTube怎么赚钱？

收益拆分

第一，广告分成机制。

创作者拿55%，平台拿45%。

RPM（千次播放收益）2-12美元，取决于受众地区、内容垂直度。

但前提：

你得先达到1000粉+4000小时播放时长，才能开通广告分成。

新手前3个月，可能一分钱都拿不到。

第二，变现路径多元。

广告分成、频道会员、小黄车、联盟营销、品牌赞助。

但每条路都有门槛。

联盟营销需要精准流量，品牌赞助需要粉丝基础。

不是"发了视频就有钱"，是"有了价值才有人买单"。

第三，算法逻辑关键。

观看时长>点击率>互动数据。

短视频看3秒完播率，长视频看平均观看时长。

但操作起来难。

知道"钩子设计"重要，不代表你能写出让人停留3秒的开头。

给想做YouTube的人几条实在建议。

第一，先验证再投入。

别一上来就买设备、学剪辑、搞矩阵。

先用手机拍3条视频，发出去看反馈。

有人看、有人评、有人追，再考虑放大。

第二，原创比搬运稳。

搬运+翻译，短期可能起量，长期必被限流。

哪怕只是"二次创作"，也要加自己的观点、剪辑、包装。

平台要的是"增量价值"，不是"内容搬运工"。

第三，算清楚时间成本。

一条视频从选、拍摄、剪辑、发布、优化，至少3-5小时。

如果RPM是3美元，1万播放才赚30美元。

时薪算下来，可能不如打工。

第四，合规是底线。

版权、音乐、肖像、广告披露。

这些规则，平台写得清清楚楚。

别为了省事前踩雷，后期申诉更难。

第五，别指望"快速起号"。

4000小时播放时长，不是靠刷量能解决的。

平台能识别异常流量，刷了反而封号。

踏实做内容，比研究"骚操作"更重要。

最后说句实。

别人起号，不代表你也能。

每个人的内容、节奏、资源都不一样。

别把"别人能"当"我也能"。

别把"资料包"当"成功秘籍"。

别把"平台红利"当"个人能力"。

2、用了下cursor以后，感觉这个编程软件有点反人类，不知道是不是我是设计出身，真心觉得对小白不够友好，所以用的一直是aistudio，毕竟去年谷歌还是大善人，但是联调就只能一直手动，非常笨

3、方向选择我看了下大部分有搞导航站的，也没有别的好想法，于是第一个也是导航站，真是小白啥也不懂就瞎搞…数据是写的脚本扒拉的竞品然后又改的，当时也不懂json，而且那时候是第一次开始频繁的使用学习ai对话，正是这样才发现了ai胡说八道，被骗了好几次…因为ai老是给错误的方向小白全当真了

4、因为这次熟练掌握了和ai对话输入需求获得结果的能力，算是很大的进步吧，后面通过ai来学习别的真的是事半功倍，完全归功于反复给ai说需求不断试错调整那段经历

5、导航站发上去还真有很多来让我收录的，但是那时候不懂内链结构。没有什么单页工具，一个导航才几十个页面，批量做单页又觉得太费精力了，哪像现在codex一句话的事…放上去就没管了

6、后面又做了时钟站，完全是随手让ai写了个页面发现这个需求好简单，但是从来没现站在seo的角度去考虑词的难度问题，算是结结实给小白上课，加上我发现倒计时页面比首页流量大，于是犯了很大的错误就是把首页全改成倒计时页面，流量直接归零了，这也是我被ai骗得一个地方，现在让我改动h1和title我真的不会乱改了

7、中间做了翻页时钟，因为发现竞品流量出奇的大，但是当时也没注意这个东西虽然kd好做，就是很难挂上adsense…这也是一个大坑，这个站靠疯狂上外链，一个月以后kd从0到了20多，曝光和点击也是上升了很多，也是让我知道外链的威力

8、因为5-7月又要搬家又要收拾又要上站，每天晚上兴奋搞到1点多，直接腰椎间盘突出。我就决定8月开始休息不干了，直接让我错过了8月份新词方向…当然即使知道也估计不会做，这就是为什么要有个好老师跟着的原因，建议小白都去看看出海大神们的各种教程

9、后面就是工作非常的忙，除了一点点维护，可以说完全没精力搞了，被工作榨干了，而且工作搞得是漫剧…和上站是完全两个东西，从去年10月到4月底离职期间，可以说基本上没怎么搞，也估计错过了不少

10、5月帮朋友忙项目去了，6月到目前算是全职搞了有半个月吧，又多上了一些站，也多看了很多，发现了很多之前自己的想法错误，比如现在一开始仍然觉得上带新词的域名就能有流量，上个image2什么的😂，但是类似香蕉，即梦这种的 流量大词其实都被占光了;可能除了游戏词，还有从来没出现过的工具词。才会好做吧？也看清了导航站的玩法，还有自己的一些其他站为什么没流量，上站性价比问题等等，多上站才能看清很多东西

11、从今天来看虽然进场不算早，但是绝对不算晚，我也不算纯小白，只是验证了很多东西，却没有拿到结果

12、写出来也是想算是一个可以供人参考的经历吧，希望大家都能出海顺利，赚到美刀
